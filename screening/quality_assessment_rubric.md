# Quality Assessment Rubric

The included studies were assessed using five quality assessment items: QA1, QA2, QA3, QA4, and QA5. Each item was scored using the values available in the study selection file.

## Scoring format

- `1.0` = criterion fully satisfied
- `0.5` = criterion partially satisfied
- `0.0` = criterion not satisfied or not reported

## Total score

The `quality_score` is calculated as the sum of QA1–QA5. In the current dataset, the included studies satisfy the minimum eligibility threshold used in the review.
